# 個人投資規劃系統（試作版）

這個試作版包含兩個獨立入口，兩者共用同一套計算模型與瀏覽器儲存資料。

- `web/index.html`：桌面／平板儀表板
- `mobile/index.html`：手機專用操作介面

## 試作版計算假設

- 規劃自 2026 年 6 月開始，2026 年薪資與定期投資均按 6 個月計。
- 2027 年起薪資按每年 14 個月計。
- 2026 至 2027 年固定投入 0050 每月 70,000 元、VOO 每月 200 美元、NVDA 每月 600 美元。
- 2028 年起保留現金目標後，再依年齡對應配置投入。
- 購車貸款以年利率 2.5%、48 期本息平均攤還。
- 頭期款優先出售 00919，差額才由現金支應，但現金不得低於 800,000 元；若仍不足，報表會列出資金缺口。
- 薪資計入至 64 歲，65 歲當年起停止計入。
- 61 歲當年清空槓桿部位，並估算出售 NVDA 後轉入 0050 與高股息部位。
- 報價按鈕在瀏覽器直接嘗試讀取 Yahoo Finance 公開圖表資料；若跨網域限制導致失敗，會保留上次資料。正式部署建議加上受控的後端代理。

## 本機預覽

使用任一靜態網站伺服器，以專案根目錄作為網站根目錄，再開啟：

- `/web/`
- `/mobile/`
